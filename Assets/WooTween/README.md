# WooTween
* 一款简单好用的动画插件
* 支持插值模式 （ Normal, Shake, Punch, Jump, Bezier, Array）
* 支持 串行、并行
* 提供常见扩展方法  （Transform、Camera、UGUI、Recttransform 等）
* 支持自定义扩展方法
* 支持编辑器预览
* 支持 DoPath Scene 编辑
* 支持采样 可以与[ActionEditor](https://github.com/OnClick9927/ActionEditor)配合使用（实现类似于unity animation 窗口效果）
